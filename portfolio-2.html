About this report

Main Page (General Results)

This one was developed focused to bring more details to the investors related Oi Company results. The data that the business share with all investors is available to be access by csv file, so then I did create an python script to clean this data and load the useful data into my sql server db that was created for this project in specifically.

In the first graph you will see a relation between percentage of net interest and the consolidated interest. All values here is presented by quarter.

In the second chart we can see a general overview, also by quarter for profit vs loss.

And below we can check two round charts with more details about income per category and Costs per category respectively.

Main Page (Tool Tip)

Keep in this main page (seconde image) I added one tooltip for income per category where the customer can check total of income for each product/service.

Third Image

In this last image we can see debt data, once that this company (OI) is under judicial recovery to avoid bankruptcy.

For the first chart I'm presenting inside of bar a comparison between long term debt and short term debt. And into the line the net debt.

Next to this chart mentioned above we have also a graph of bar showing how this debt was growing by quarter since 2020.

Below we have a line comparison showing the sum of net debt by quarter vs national currency exposure and international currency exposure. To finish in the round chart we have total of net debt per year since 2020.

Technologies that were applied

Power BI:

The main tool used to develop this report, inside of this one I could do also build all ETL process and data cleansing. So when I load again new data (from my data wharehouse to power bi) all ETL proccess is completly done to support the standard data, and transform it in data that I really need.

Excel (CSV) & SQL Server:

Excel was my reference with standard data extracted from company RI portal. The all data that I really need was transformed and loaded into my sql server database.

Python:

In python I did use only to transform the CSV file for a data that I really needed to build this analysis in power bi. So in future if I need to extract new standard data, this one will be transformed and loaded using my python script, just running a simple command line. (Of course this one can be change to a job that can run daily or monthly).
